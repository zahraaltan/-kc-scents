#KC Scents 🕌

A luxury perfume e-commerce website built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Site:** [zahraaaltan.github.io/kc-scents](https://zahraaaltan.github.io/kc-scents)

---

## Features

- 🛍️ **Product Catalog** — Browse a curated collection of luxury fragrances
- 🛒 **Cart System** — Add, remove, and update quantities using localStorage
- 💳 **Checkout Page** — Complete order form with real-time validation
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 🎥 **Video Hero Section** — Immersive homepage with background video

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript (localStorage, DOM Manipulation)

---

## Pages

- `index.html` — Homepage with hero video and product showcase
- `shop.html` — Full product listing
- `cart.html` — Cart with quantity management
- `checkout.html` — Order form with validation
- `about.html` — Brand story
- `contact.html` — Contact form

---

## What I Learned

- DOM manipulation and dynamic rendering with JavaScript
- localStorage for persistent cart data
- Form validation with regex
- Responsive design with media queries
